## 项目 Agent 详细规则

本文件是根目录 `AGENTS.md` 引用的详细规则说明。根目录 `AGENTS.md` 是入口文件；如果两者存在冲突，以根目录 `AGENTS.md` 为准。

### 工作规则

- 默认先阅读相关代码和文档，再开始修改。
- 修改前先说明影响范围。
- 只做和当前任务相关的最小改动。
- 提交前必须运行最小验证。
- 说明用中文，代码、命令、文件名保持英文。
- 不要修改无关文件。
- 不要提交无关的未跟踪文件或工作区改动。

### 流程引用

- 多文件任务：遵循 `doc/PLAN.md`。
- 任务完成总结：遵循 `doc/EndOutput.md`。
